# ❤️ Kardioku App

Kardioku adalah Aplikasi Deteksi Dini Risiko Penyakit Kardiovaskular berbasis Algoritma Hybrid Machine Learning. Aplikasi ini dirancang untuk membantu pengguna memantau kesehatan jantung mereka secara mandiri dengan antarmuka yang modern dan ramah pengguna.

## ✨ Fitur Utama
* **Analisis Cerdas:** Memprediksi tingkat risiko kardiovaskular menggunakan algoritma *Machine Learning* (berbasis Framingham Heart Study).
* **Riwayat & Pelacakan:** Memantau tren tekanan darah (sistolik & diastolik) dari waktu ke waktu.
* **Manajemen Profil:** Personalisasi data fisik (umur, jenis kelamin, tinggi, dan berat badan) untuk akurasi prediksi.
* **Privasi Maksimal:** Data pengguna dienkripsi dengan standar keamanan tinggi (Bcrypt) dan disinkronkan secara aman.
* **Notifikasi & Pengingat:** Fitur pengingat harian untuk melakukan pemeriksaan rutin.

## 🛠️ Teknologi yang Digunakan
* **Frontend:** React, Vite, Tailwind CSS, TypeScript
* **Backend:** Python, FastAPI, SQLAlchemy
* **Database:** MySQL (Aiven Cloud)
* **Machine Learning:** Scikit-Learn
